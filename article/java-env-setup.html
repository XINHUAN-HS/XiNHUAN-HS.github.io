☕ Java 环境变量配置详解（JDK & Maven） — 2026-06-18

Java 开发离不开正确的环境变量配置。本文将带你一步步配置 JAVA_HOME、PATH 和 MAVEN_HOME。

1. 下载并安装 JDK

前往 Adoptium 下载 OpenJDK LTS 版本（如 JDK 17 或 21）。安装时记住安装路径，例如：

  • Windows: C:\Program Files\Eclipse Adoptium\jdk-17.0.9.9-hotspot
  • macOS/Linux: /usr/lib/jvm/jdk-17 或 /Library/Java/JavaVirtualMachines/jdk-17.jdk/Contents/Home

2. 设置 JAVA_HOME

Windows

# 系统变量中添加
变量名: JAVA_HOME
变量值: C:\Program Files\Eclipse Adoptium\jdk-17.0.9.9-hotspot

# 然后在 PATH 中添加
%JAVA_HOME%\bin

macOS / Linux

# 编辑 ~/.bashrc 或 ~/.zshrc
export JAVA_HOME=/usr/lib/jvm/jdk-17
export PATH=$JAVA_HOME/bin:$PATH

# 使生效
source ~/.bashrc

3. 验证配置

java -version
javac -version
echo $JAVA_HOME   # macOS/Linux
echo %JAVA_HOME%  # Windows

4. 配置 Maven 环境变量

下载 Maven 并解压，然后设置 MAVEN_HOME 和 PATH。 

# 设置 MAVEN_HOME
export MAVEN_HOME=/opt/apache-maven-3.9.6
export PATH=$MAVEN_HOME/bin:$PATH

# 验证
mvn -version
⚠️ 注意：确保 JAVA_HOME 指向 JDK 的根目录（包含 bin、lib 等子目录），而不是 JRE 目录。

5. 多版本 JDK 管理

推荐使用工具如 jEnv（macOS/Linux）或 SDKMAN! 轻松切换 JDK 版本。 

# 使用 SDKMAN! 安装和管理
curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"
sdk list java
sdk install java 21.0.1-tem
sdk use java 21.0.1-tem
