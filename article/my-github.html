2. XiNHUAN-HS.github.io —— 深空主题静态博客

这个项目就是本篇所在的站点 😄。核心卖点是"深空美学 + 极简"—— Canvas 画几百颗星 + 漂移星云 + 太阳系行星点缀，配一个数字时钟和交互式日历。

  • 视觉：纯黑底 + 银河光带，STAR_COUNT = 460 可调
  • 文章系统：articles.json 驱动，iframe 内嵌，支持搜索 + 日期排序
  • 自动化：create_article.py 扫 article/ 目录自动抽标题/日期/导语，回写 JSON
  • 部署：GitHub Pages，根目录 .github/workflows 管 CI

整套东西没用框架，HTML + CSS + Vanilla JS 硬写，配色和模板就是你正在看的这份。

🔗 github.com/XINHUAN-HS/XiNHUAN-HS.github.io

🌐 xinhuan-hs.github.io

✅ 小技巧：新文章写完直接 python create_article.py， 元数据就进 articles.json 了，不用手改。

3. XinHuanNewTab —— 极简多引擎搜索新标签页

Manifest V3 写的浏览器扩展，Chrome / Edge 都能装。 定位就是"打开新 tab 别给我堆新闻，给个干净搜索框就行"—— 支持多引擎切换，UI 走简洁路线，图标 16/32/48/128 四档齐了。

  • manifest.json —— MV3 清单
  • index.html + style.css + main.js —— 主页面 / 主题变量 / 搜索切换
  • background.js —— 后台服务

装法就是扩展管理里开开发者模式 →"加载已解压的扩展程序"指到文件夹，零依赖。

🔗 github.com/XINHUAN-HS/XinHuanNewTab

4. XinHuanToolKit —— 终端式赛博工具箱

这一款味道最重，Cyberpunk 风 + 模拟终端操作感。页面分四区：

  • 左侧：系统监控面板 —— CPU / 内存 / 磁盘折线图 + 网络拓扑 + 上下行速度
  • 中央：终端交互区 —— 多视图切换（本地终端 / Iframe 应用），模拟 Shell 输入
  • 右侧：文件系统 —— 应用搜索 + 分类入口
  • 底部：虚拟键盘 —— 全布局，Ctrl/Alt/Shift/CapsLock 都齐，点按输终端

当前监控数据是前端模拟的，script.js 里定时器换成 fetch('/api/system') 就能接真数据。 新应用往 apps.json 加，create_apps.py 会根据配置生成 apps/ 目录和对应页面。

🔗 github.com/XINHUAN-HS/XinHuanToolKit

🌐 xinhuan-hs.github.io/XinHuanToolKit

💡 提醒：ToolKit 的 Iframe 应用视图要确保目标地址允许 CORS； backdrop-filter 在旧版浏览器也可能掉帧。
